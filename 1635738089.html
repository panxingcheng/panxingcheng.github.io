# Eclipse

# Eclipse 设置

# 主题

Window -> Preferences -> General -> Appearance

  • Theme -> Dark

# 字体大小

Window -> Preferences -> General -> Appearance -> Colors and Fonts

  • Basic -> Text Font -> 大小 -> 12（小四）

# 文件内容类型编码格式

Window -> Preferences -> General -> Content Type

  • Text -> Default encoding -> UTF-8

  • Text -> Java Properties File -> Default encoding -> UTF-8

  • Text -> JSP -> Default encoding -> UTF-8

# 关闭前确认

Window -> Preferences -> General -> Startup and Shutdown

  • Confirm exit when closing last window

# 工作空间编码格式

Window -> Preferences -> General -> Workspace

  • Text file excoding -> UTF-8
